  5. What is the difference between semantic and non-semantic elements?

    We say that an element is semantic if the tag enclosing it carries meaning (such as the tag <article> which signifies that the element is an article). In contrast an element is said to be non-semantic if no such meaning is attached to its tag (such as the tag <span> which only defines a span of a text as an element without indicating why).

  6. Why do we prefer using semantic elements?

    For the purposes of accessibility and SEO.
